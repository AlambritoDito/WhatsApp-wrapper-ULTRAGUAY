# Contributing to WhatsApp Wrapper Ultra

Thank you for your interest in contributing to `@brito/whatsapp-wrapper-ultra`! We welcome contributions from the community to help make this project better.

## How to Contribute

### Reporting Issues
If you find a bug or have a feature request, please open an issue on GitHub.
- **Bugs**: Describe the issue in detail, including steps to reproduce, expected behavior, and actual behavior.
- **Features**: Explain the new feature and why it would be useful.

### Submitting Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies**: `npm install`
3. **Make your changes**. Ensure your code follows the existing style.
4. **Run tests**: `npm test` to ensure your changes don't break anything.
5. **Lint your code**: `npm run lint`
6. **Commit your changes**: Use clear and descriptive commit messages.
7. **Push to your fork** and submit a Pull Request.

### Coding Standards
- We use **TypeScript**.
- We use **ESLint** and **Prettier** for linting and formatting. Please run `npm run format` before committing.
- Write meaningful variable and function names.
- Add comments where necessary to explain complex logic.

## Development Setup
1. Clone the repo.
2. Copy `.env.example` to `.env` and fill in your credentials.
3. Run `npm run dev` to start the development server.

Thank you for your contributions!
